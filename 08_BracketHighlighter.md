# BracketHighlighter
- {},[],<>,"",'',() 와 같이 열리고 닫히는 기호가 쌍으로 존재해야 하는 코드를 하이라이트처리해주는 확장기능
```
{
"bracket_styles": {
"angle": {
"color": "entity.name.class",
"style": "outline"
},
"tag": {
"color": "entity.name.class",
"style": "outline"
},
"default": {
"color": "entity.name.class",
"style": "solid"
}
}
}
```
- 참고 https://github.com/facelessuser/BracketHighlighter
